== Requirements Class "Filtering Zone Data with CQL2"

=== Overview

The OGC API - DGGS "Filtering Zone Data with CQL2" requirements class defines a `filter` query parameter
where an OGC Common Query Language (CQL2) expression can be used to filter the data returned for a zone,
for example based on the properties / fields of the data, or for performing spatial and/or temporal intersections.
