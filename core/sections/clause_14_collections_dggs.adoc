== Requirements Class "Collection DGGS"

=== Overview

The OGC API - DGGS "Collection DGGS" Requirements Class defines the availability of DGGS resources applying to one or more collections of geospatial data, as defined by _OGC API - Common - Part 2: Geospatial data_.
