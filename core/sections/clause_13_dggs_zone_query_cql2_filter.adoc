== Requirements Class "Filtering Zone Queries with CQL2"

=== Overview

The OGC API - DGGS "Filtering Zone Queries with CQL2" requirements class defines a `filter` query parameter
where an OGC Common Query Language (CQL2) expression can be used to filter the list of returned zones.
Such CQL2 queries enable filtering based on the properties / fields of the data, or on spatiotemporal intersections.
